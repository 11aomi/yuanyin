## 使用说明

1. **安装要求**
   - Android 5.0 (API 21) 或更高版本
   - 启用"未知来源"应用安装权限
   - 确保设备有足够存储空间

2. **安装方法**
   ```bash
   # 使用adb安装
   adb install <version>-debug.apk
   ```

3. **测试注意事项**
   - Debug版本可能包含调试信息和日志
   - 性能可能不如Release版本
   - 建议在测试设备上使用
